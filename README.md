# organizador_fotos
Como fotógrafio acumulei muitas fotos ao longo dos anos, com diversas cópias em diversos dispositivos de memória distintos. Este código vem solucionar a organização, porém com várias proteções que visão não perder fotos que movimentação ou por problemas na identificação. Ele organizar por AAAAMM, separando o que está em estado RAW e editada.
